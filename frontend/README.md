# Wozniak Bau Website

Astro + Tailwind + GSAP landing page for Robert Jan Wozniak Dienstleistungen am Bau.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Notes

- Public assets live in `public/assets/`.
- For GitHub Pages project sites, set `site` and `base` in `astro.config.mjs` before deploying.
